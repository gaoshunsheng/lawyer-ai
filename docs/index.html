案由: 劳动争议 程序: 二审 法院: 上海市第一中级人民法院 日期: 2023-08-15
争议焦点
用人单位以劳动者严重违纪为由解除劳动合同，但仅有口头警告记录，是否构成违法解除
裁判要点
法院认为，用人单位主张劳动者严重违纪，应当承担举证责任。本案中，公司仅提供口头警告记录，无书面规章制度、无违纪通知送达证据，不足以证明劳动者存在严重违纪行为，应承担举证不能的不利后果，构成违法解除。
